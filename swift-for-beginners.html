Variables and Types

Welcome to Swift for Beginners

A variable is a fundamental unit of programming which we use to assign a name to a value. Variables allow us to store information so we can use it later when we need it.

Intro to Strings

A strings are a sequence of characters.

String Initialisation

Very often we'll initialise strings using string literals. let myString = "A string" var myEmptyString = "" var yetAnotherString = String()

Properities

The characters property holds the collection of characters that make up a string. The property can be used to loop through all of the characters in a string.

var name = "Ash" for character in name.characters { print("\(name)") }

I have a question. Is this collection an array?

It's worth noting that you can get the number of characters in a string using the character property's count property.

var name = "Ash" var charCountInName = name.characters.count

You can also reverse a string.

var name ="Ash" var reversedName = name.characters.reversed()

It's also possible to get the unicode scalars.

var name = "Ash" var scalars = name.unicodeScalars
