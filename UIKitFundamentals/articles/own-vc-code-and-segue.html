In storyboard, Apple refers to the transition between view controllers as a segue. These are shown in storyboard with a line between two storyboards with a circle in the middle.

We have learned to use just code to display a storyboard view controller. Now we're going to combine code with a storyboard segue.

In storyboard, open the popup menu you get from either view controller icon. Under triggered segues drag manual to view controller you want to display. At this point you'll get a few options for the segue, choose modal here.

You should now set the storyboard identifier for the segue. Select it in the canvas, go to attributes inspector and give it an identifier. 

                @IBAction func rollDice() {
                    performSegueWithIdentifier(identifier: "rollDice", self)
                }
